# Helpful Links & Project Resources

This document provides a centralized list of links to official documentation, guides, and other helpful resources for the technologies used in the AdventureNest project.

## 1. Core Framework & Language

### Next.js (with App Router)
*   **Official Documentation:** [nextjs.org/docs](https://nextjs.org/docs) - The primary source for all Next.js features.
*   **App Router Guide:** [nextjs.org/docs/app](https://nextjs.org/docs/app) - Essential reading for understanding the new App Router paradigm.
*   **API Routes:** [nextjs.org/docs/app/building-your-application/routing/route-handlers](https://nextjs.org/docs/app/building-your-application/routing/route-handlers) - For building backend API endpoints.
*   **Server Actions:** [nextjs.org/docs/app/building-your-application/data-fetching/server-actions-and-mutations](https://nextjs.org/docs/app/building-your-application/data-fetching/server-actions-and-mutations) - For server-side data mutations directly from components.

### React
*   **Official Documentation:** [react.dev](https://react.dev/) - The new, official React documentation with a focus on functional components and hooks.

### TypeScript
*   **Official Handbook:** [typescriptlang.org/docs/handbook/intro.html](https://www.typescriptlang.org/docs/handbook/intro.html) - The definitive guide to the TypeScript language.
*   **TypeScript with React:** [cheatsheet.for-react.dev/](https://cheatsheet.for-react.dev/)- A helpful cheat sheet for using TypeScript in React projects.

## 2. Styling

### TailwindCSS
*   **Official Documentation:** [tailwindcss.com/docs](https://tailwindcss.com/docs) - Comprehensive documentation for every utility class.
*   **Installation with Next.js:** [tailwindcss.com/docs/guides/nextjs](https://tailwindcss.com/docs/guides/nextjs) - The official guide for setting up TailwindCSS in a Next.js project.
*   **Tailwind UI:** [tailwindui.com](https://tailwindui.com/) - Component library from the makers of Tailwind CSS for inspiration and high-quality component designs (some free, some paid).

## 3. Database & ORM

### Prisma
*   **Official Documentation:** [prisma.io/docs](https://www.prisma.io/docs/) - The main documentation for the Prisma ORM.
*   **Prisma with Next.js:** [prisma.io/docs/guides/framework-guides/nextjs](https://www.prisma.io/docs/guides/framework-guides/nextjs) - A specific guide for integrating Prisma into a Next.js application.
*   **Data Modeling:** [prisma.io/docs/concepts/components/prisma-schema](https://www.prisma.io/docs/concepts/components/prisma-schema) - Guide to defining your database schema with Prisma.

### PostgreSQL
*   **Official Documentation:** [postgresql.org/docs](https://www.postgresql.org/docs/) - The official, in-depth documentation for PostgreSQL.

## 4. Authentication

### Next-Auth.js
*   **Official Documentation:** [next-auth.js.org](https://next-auth.js.org/) - The primary source of information for setting up authentication.
*   **App Router Integration:** [next-auth.js.org/getting-started/for-app-directory](https://next-auth.js.org/getting-started/for-app-directory) - Crucial guide for using Next-Auth with the Next.js App Router.
*   **Prisma Adapter:** [next-auth.js.org/adapters/prisma](https://next-auth.js.org/adapters/prisma) - For connecting Next-Auth to your PostgreSQL database via Prisma.

## 5. Payment Gateway

### Stripe
*   **Official Documentation:** [stripe.com/docs](https://stripe.com/docs) - The entry point for all Stripe APIs and products.
*   **Accept a Payment Guide:** [stripe.com/docs/checkout/quickstart](https://stripe.com/docs/checkout/quickstart) - A quickstart guide for creating a Checkout session.
*   **Stripe with Next.js (App Router):** [github.com/vercel/next.js/tree/canary/examples/with-stripe](https://github.com/vercel/next.js/tree/canary/examples/with-stripe) - Vercel's official example of a Stripe integration.
*   **Stripe Connect:** [stripe.com/docs/connect](https://stripe.com/docs/connect) - For handling C2C payments in the Community Marketplace.

## 6. Deployment

### Vercel
*   **Official Documentation:** [vercel.com/docs](https://vercel.com/docs) - Documentation for the Vercel platform.
*   **Deploying a Next.js App:** [vercel.com/docs/frameworks/nextjs](https://vercel.com/docs/frameworks/nextjs) - A step-by-step guide for deploying your project.
*   **Environment Variables:** [vercel.com/docs/projects/environment-variables](https://vercel.com/docs/projects/environment-variables) - How to manage secrets and configuration for production.
